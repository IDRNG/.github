# IDRNG - Next-Generation Inter-Domain Routing Architecture and Protocols Research Group

## About

This is the repo for IDRNG-related materials.

IDRNG is short for Next-Generation Inter-Domain Routing Architecture and Protocols Research Group.

## IDRNG Related Documents

A list of pointers to documents that are related to IDRNG can be found [here](https://github.com/IDRNG/IDRNG-Related-Materials).
